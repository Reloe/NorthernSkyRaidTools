## Changelog

- Add new option for Taunts. When enabled the spellid will automatically change to the equivalent taunt-spell of the class the player is currently on.
- Add option for Memory Game to show the name of the sender on the display for problem solving infight
- Newly created Lura-Macros will now the name of the raidmarker into raidchat instead of numbers. The numbers will continue working to not break existing macros.
- Add setting to always use TTS over soundfiles unless a soundfile was specifically selected(previously it would use soundfiles if it found a match with that name)
- On Ready Check send an alert to the player if they are missing the weekly stacking raidbuff for the equivalent raid they are zoned into
